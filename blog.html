How to Install GCam on Nothing Phone (2) - The Ultimate Guide

5 MIN READ ADMIN

The Nothing Phone (2) comes with impressive hardware, utilizing the Sony IMX890 sensor. However, many enthusiasts prefer the image processing of Google Camera (GCam) for better dynamic range and night sight capabilities.

Why Use GCam?

Google's computational photography algorithms can extract more detail from the Nothing Phone sensors. Features like HDR+ Enhanced and Astrophotography Mode work exceptionally well with the transparent back glyph interface used as a soft fill light.

Installation Steps:

  1. Download the latest BSG or AGC GCam APK from our "Playground" section.
  2. Move the Config XML file to the /Configs/ folder in your internal storage.
  3. Open the app and double-tap the black area near the shutter button.
  4. Select the XML file tailored for Nothing OS 2.5 and hit "Import".

Once installed, you will notice significantly less noise in low-light shots and more natural skin tones in portraits.
